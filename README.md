### ❄️ 部署流程
1）进入项目目录，终端执行命令（自动安装完项目依赖，生成node_modules文件夹）：<br>
  ``` npm i ```<br>
2）终端输入命令，来启动项目（如果没有部署好后端项目会是空白的（后端项目请见文章最后面，非常简单））：<br>
   ``` npm run dev ``` <br>
3）如果是想部署到云服务器上，需要先生成dist文件夹，并且进行压缩：<br>
   ``` npm run build ```<br>
